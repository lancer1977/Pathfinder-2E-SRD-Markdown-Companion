# Feature Documentation

This directory documents all features and roadmap items. All state is tracked in front matter.

## Front Matter Schema

```yaml
---
title: <human readable>
status: backlog | spec | doing | review | done | roadmap | deprecated
owner: @username
priority: critical | high | medium | low
complexity: 1-5
created: YYYY-MM-DD
updated: YYYY-MM-DD
tags: [<topic>, <component>]
---
```

## Status Legend

| Status | Meaning |
|--------|---------|
| `backlog` | Not yet planned or started |
| `spec` | Requirements being defined |
| `doing` | Currently being implemented |
| `review` | Implementation complete, pending review |
| `done` | Fully implemented and verified |
| `roadmap` | Future/planned feature |
| `deprecated` | No longer maintained |
