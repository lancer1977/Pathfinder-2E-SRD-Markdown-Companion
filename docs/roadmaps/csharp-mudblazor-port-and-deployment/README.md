Deliver a structured C# rewrite of the original Python RAG chat UI with deployment automation and repeatable corpus refresh.

## Scope

- Blazor Web App with MudBlazor
- SQLite persistence
- Ollama integration
- markdown corpus refresh from GitHub
- helper scripts for local and deployment workflows
- GitHub Actions build/deploy support
- Docker image deployment
